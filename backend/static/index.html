            Interfaces with error in status
            
    
            Status contains err (case-insensitive), e.g. errdisabled, err-disable, Error. Recover and clear counters are only offered for Leaf devices on host ports Ethernet1/1–Ethernet1/48 (or 1/1–1/48); other interfaces (e.g. spine uplinks) show actions disabled. Recover runs configure, interface, shutdown, no shutdown. Clear counters runs clear counters interface <name> in privileged exec. Requires username/password credential (not API-key-only).

            Help
            
  
            This panel helps you run pre/post checks, look up NAT and BGP, find leaves, and manage inventory. All tables support sorting and export (CSV/Excel) where indicated.
            

  
            Home
            
  
            The dashboard shows all features in a 3×3 grid. Use the menu (hamburger icon) or cards to navigate.
            
  
            

  
            Navigation
            
  
            Use the menu (hamburger icon) to switch between pages. The top event bar shows counts of successful operations, warnings, and errors; click a counter to see the event list. Use Home / Back in the header to return to the dashboard or go back.
            
  
            

  
            Pre/Post Check
            
  
            Capture device state before and after a change, then compare results.
            
  
              
    
            • Choose Fabric, Site, and Phase (e.g. Pre-Check, Post-Check, or Custom Command).
              • 
    
            • Select devices and click Run. Pre-Check creates a saved report; Post-Check compares against a chosen Pre-Check report.
              • 
    
            • On the results page you can save the report with a custom name, export as ZIP (HTML with embedded styles), and view interface consistency (groups with different status across devices).
              • 
    
            • Saved reports appear in the dropdown on the Pre/Post page: open, delete, or run Post-Check against them.
              • 
    
            • Only show and dir commands are allowed; the app blocks commands that could change device state.
              • 
  
            
  
            
  
            

  
            NAT Lookup
            
  
            Find NAT rule and translation for a source/destination pair using Palo Alto.
            
  
              
    
            • Enter Source IP and Destination IP (or use defaults), then run the lookup.
              • 
    
            • Results show the translated IP, rule, and BGP path for the first translated prefix. Use Open in BGP page to jump to BGP with that prefix pre-filled.
              • 
  
            
  
            

  
            Find Leaf
            
  
            Locate which leaf switch has a given IP in the fabric.
            
  
              
    
            • Select Fabric and Site, enter the IP, and run. The table lists matching device(s) with hostname, fabric, site, and role.
              • 
  
            
  
            

  
            BGP / Looking Glass
            
  
            Look up prefix or AS via RIPEStat: routing status, RPKI, visibility, history diff, and per-prefix AS paths from one router.
            
  
              
    
            • Enter an IPv4 prefix (e.g. 1.1.1.0/24) or an AS number (e.g. 9121 or AS9121), then run the lookup.
              • 
    
            • Per-prefix paths: For each prefix, the app shows the best two AS paths from a single RIS route collector (RRC). Paths are drawn as router icons; consecutive duplicate ASes are collapsed (e.g. 5 × AS9121).
              • 
    
            • WAN RTR: When querying by AS, the app also checks your WAN routers for router bgp <AS> in running-config and lists matches with Fabric and Site.
              • 
    
            • Use Looking Glass and BGP play tables for peer-level detail and path changes over time.
              • 
  
            
  
            
  
            
  
            

  
            REST API
            
  
            Run single or multi eAPI (Arista) requests on selected devices. Select fabric, site, hall, and role to list devices, then choose devices and enter commands (one per line). The first line can be enable with password. Results are shown in a table; use examples (e.g. version, running-config) to get started.
            

  
            Transceiver Check
            
  
            View SFP/optics DOM information (temperature, power) per interface on Arista EOS and Cisco NX-OS. Select fabric, site, hall, role, then devices, and run the check.
            
  
            

  
            Credential
            
  
            Store and manage login credentials for inventory devices. Credentials are encrypted and referenced by name in the inventory (Credential column). Add, edit, or delete entries from this page.
            
  
            

  
            DCI / WAN Routers
            
  
            Compare route-map configuration on Arista DCI or WAN routers. Select scope (DCI or WAN), pick devices, then Compare to build a table of peer groups, route-maps, and prefix-lists. Use Search by prefix to find which route-maps reference a given prefix.
            
  
            

  
            Inventory
            
  
            Manage the device list used by all other pages: hostname, IP, fabric, site, hall, vendor, model, role, tag, and credential. Use Add device, Edit, Import (CSV), and Export. Column visibility can be toggled via Columns.
            
  
            

  
            Live Notepad
            
  
            One shared plain-text notepad for everyone. Open Live Notepad from the menu; type in the text area. Changes are saved automatically and synced every few seconds. Everyone sees the same content; no formatting, just text.
            

  
            Diff Checker
            
  
            Compare two texts like Git/GitHub diff. Paste in Left (before) and Right (after), then click Diff check. Matching lines are found automatically (LCS), so deleting one line does not mark everything below as changed. Table: − removed (red, left column only), + added (green, right column only). Left # and Right # are line numbers in each file.
            

  
            Subnet Divide Calculator
            
  
            Enter network and mask, click Update. Table shows one row per subnet. Divide splits a row into two (prefix+1). Join column: grey cells merge subnets back; Reset clears divisions.
            

  
            Tables and export
            
  
            Tables use DataTables.net: sort by column, paginate, and use CSV/Excel export where the buttons are shown. Pre/Post diff tables use green for added, red for removed, and amber for modified values.
            

  
            Theme
            
  
            Use the theme toggle in the header to switch between dark and light mode. The choice is stored in your browser.
            
  
            

  
            
  
            REST API
            
  
            Select fabric, site, hall and role to list devices. Currently: Arista eAPI; Cisco and Palo Alto support coming later.
